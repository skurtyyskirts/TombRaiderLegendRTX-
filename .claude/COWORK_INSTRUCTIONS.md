This is the TombRaiderLegendRTX project — a D3D9 FFP proxy DLL (NightRaven1) making
Tomb Raider: Legend compatible with NVIDIA RTX Remix.

Key directories:
- proxy/         — proxy DLL source code
- retools/       — offline static analysis tools
- livetools/     — Frida-based live tracing
- autopatch/     — autonomous patch-test loop
- docs/status/   — WHITEBOARD.md (live status), TEST_STATUS.md (build results)
- TRL tests/     — archived builds with screenshots, logs, source snapshots

When creating documents: use markdown. Include addresses in monospace. Date everything.
All findings go into CHANGELOG.md.
